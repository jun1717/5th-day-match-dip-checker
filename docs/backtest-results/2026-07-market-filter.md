# 検証記録: 地合い(市場レジーム)フィルター (2026-07-13)

`docs/plan-market-and-earnings-filter.md` のM5(地合いフィルターのA/B検証と既定値決定)の実施記録。

## 検証環境

- 価格データ: `data/prices_backtest.csv`(2年分・分割調整済み・確定日足。2026-07-13取得。市場指標1306を含む486営業日)
- 対象期間: 2025-07-28 〜 2026-06-15(214営業日)、対象status: buy_candidate、stopMode: prev-day、maxHoldDays: 30
- rulesHash: `2764dd04e845`(marketIndexCode=1306 / sizingMode=risk / stopTight・volumeフィルター=flag / earningsFilterMode=exclude ※earnings.csvは空のため不発)
- 地合い定義: 1306(TOPIX連動ETF)の終値が25日線より上 かつ 25日線が上向き → OK。期間内の内訳は **OK 178日 / NG 36日**(NGの過半は2026年3月の調整局面: 19日)。

## 判断(事前基準: 期待値R改善または同等 + 最大DD改善 + 約定数60%以上 + NG日コホート劣位)

**`marketFilterMode = "flag"` を維持**(excludeに昇格しない)。4条件のうち3条件が不成立。

解釈(なぜマスタースイッチが効かなかったか):

1. **個別・テーマ条件が実質的な地合いフィルターとして既に機能している**。買い候補は「終値>25日線・25日線上向き・テーマスコア80以上」を要求するため、市場全体がNGの日にこれを満たすのは相対的に最も強い銘柄×テーマだけ。その生き残り選抜が「NG日の候補ほど良い」という逆転を生んでいる。
2. **この期間のNGは深押しからの反発局面だった**。NG 36日の過半は2026年3月の調整(19日)で、その後相場は回復した。押し目買いにとって「指数が25日線を割った直後の強い銘柄」はむしろ絶好の仕込み場だったことになる。
3. ただし**本命のシナリオ(持続的な下落相場)はこの2年に含まれていない**。サンプルはNG日シグナル27件と薄く、平均回帰した調整しか経験していない。地合いフィルターの真価(連敗回避)はベア相場でしか測れないため、この結果は「フィルター不要」の証明ではなく「この強気期間では不要だった」に留まる。

運用上の扱い:

- flagモードの警告(`market_regime_weak`)はUI・スナップショットに残る。地合いNG時に買うかどうかは、警告を見た上での裁量判断とする(ツールは落とさない)。
- スナップショットに `marketRegimeOk` が記録され始めたので、実運用履歴が溜まったら `npm run analyze:signals` の「status×地合い」集計で再判定する。**特に次の本格的な下落相場を経験した後に必ず再検証すること**。

## 決算日フィルターについて

`earningsFilterMode = "exclude"`(既定)はリスク上限の不変条件として導入済みだが、`data/earnings.csv` が空のため本検証では不発(両構成とも同一条件)。決算日データが溜まった後、`npm run backtest -- --earnings-filter off` とのA/Bと `daysToEarnings` バンド別集計(0-3日/4-10日/>10日)で「決算またぎトレードの成績」を検証できる。

## 注意事項(結果の解釈)

- ウォッチリスト自体が後知恵選択のため絶対値は楽観的。判断は構成間の相対比較に基づく。
- 期間は約10.5ヶ月の概ね強気の単一局面。NG日サンプル(シグナル27件・コホート31件)は統計的に薄い。
- 同日stop/TP同時成立は損切り扱い(保守的)、手数料・スリッページは0。
